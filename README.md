# Air-Passenger-Data-Analysis-Project
This project analyzes historical monthly air passenger data from 1949–1960 to identify trends, seasonality, and growth patterns in air travel demand.

## 🎯 Objectives
- Analyze passenger traffic trends
- Identify seasonal patterns
- Compute descriptive statistics
- Visualize passenger distribution
- Detect long-term growth trends
- Generate business insights

- ## 🛠 Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- seaborn

---
## 📊 Visualizations
- Trend Line Chart
- Histogram
- Boxplot
- Moving Average Analysis

# 🔥 Key Insights
- Passenger traffic increased significantly over time.
- Strong seasonal travel patterns were observed.
- Passenger variability increased in later years.
- Long-term growth trend confirmed using moving averages.
